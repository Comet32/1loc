title: Check if a string is upper case
updated: '2021-10-13'
---

**JavaScript version**

```js
const isUpperCase = (str) => str === str.toUpperCase();
```

**TypeScript version**

```js
const isUpperCase = (str: string): boolean => str === str.toUpperCase();
```
