updated: '2021-10-13'

**JavaScript version**

```js
const wait = async (milliseconds) => new Promise((resolve) => setTimeout(resolve, milliseconds));
```

**TypeScript version**

```js
const wait = async (milliseconds: number) => new Promise((resolve) => setTimeout(resolve, milliseconds));
```
