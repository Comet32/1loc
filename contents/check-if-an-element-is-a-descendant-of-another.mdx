title: Check if an element is a descendant of another
updated: '2021-10-13'
---

**JavaScript version**

```js
const isDescendant = (child, parent) => parent.contains(child);
```

**TypeScript version**

```js
const isDescendant = (child: Node, parent: Node): boolean => parent.contains(child);
```
