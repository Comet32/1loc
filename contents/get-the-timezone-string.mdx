updated: '2021-10-13'
---

**JavaScript version**

```js
const getTimezone = () => Intl.DateTimeFormat().resolvedOptions().timeZone;
```

**TypeScript version**

```js
const getTimezone = (): string => Intl.DateTimeFormat().resolvedOptions().timeZone;
```

**Example**

```js
getTimezone(); // 'Asia/Saigon'
```
