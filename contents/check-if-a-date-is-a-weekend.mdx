title: Check if a date is a weekend
updated: '2021-10-13'
---

**JavaScript version**

```js
// `date` is a Date object
const isWeekend = (date = new Date()) => date.getDay() % 6 === 0;
```

**TypeScript version**

```js
const isWeekend = (date = new Date()): boolean => date.getDay() % 6 === 0;
```
