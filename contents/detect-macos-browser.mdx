updated: '2021-10-13'

**JavaScript version**

```js
const isMacBrowser = /Mac|iPod|iPhone|iPad/.test(navigator.platform);
```

**TypeScript version**

```js
const isMacBrowser: boolean = /Mac|iPod|iPhone|iPad/.test(navigator.platform);
```
